# secrets

## Description

シークレットテーブル (E2E暗号化)。  
プロジェクトで使用する共用アカウント (Twitter, GitHub等) のパスワードやAPIキーを、  
現在のグループメンバーだけが閲覧できるように暗号化して保存。  
DB漏洩してもサーバー運用者でも閲覧不可。

CREATE TABLE `secrets` (

  `group_id` char(36) NOT NULL COMMENT 'Owning group',

  KEY `fk_secrets_created_by` (`created_by`),
