# Security Policy

## Reporting a Vulnerability

To report a security issue, **do not** open a public issue.

### For External Users

Email us at **info@trap.jp** or fill a form at https://trap.jp/request.

We will acknowledge your report within 48 hours and provide a detailed response indicating next steps.

Please include as much detail as possible:
- Steps to reproduce
- Affected versions
- Potential impact
- Proof-of-concept code (if available)

### For traP Members

Please report vulnerabilities via traQ [#team/SysAd/random](https://q.trap.jp/channels/team/SysAd/random).
